# The Salad House — Website Concept

A responsive, single-page restaurant website for The Salad House in Sector 49, Gurugram.

## Included

- Story-led homepage and separate founder route
- Filterable menu explorer
- Local-first ingredient story
- Interactive illustrative nutrition builder
- Working Zomato, map, phone, email and WhatsApp links
- Responsive mobile navigation
- Original generated food and kitchen photography

## Run

Open `index.html` in a modern browser. No build step is required.

## Content note

Confirmed operational facts are based on The Salad House’s public website and company profile. The founder’s identity and biography were not publicly verified, so the founder page intentionally uses editable editorial copy rather than invented personal facts. Nutrition values are clearly labelled as illustrative estimates.

Research sources used for the site:

- Official brand site: https://thesaladhouse.co/
- Official company profile: https://in.linkedin.com/company/the-salad-house-co
- Current menu listing: https://www.zomato.com/ncr/the-salad-house-sector-49-gurgaon/order

The public sources confirm fresh daily preparation, a 100% vegetarian kitchen, no overnight pre-cut vegetables, in-house components, Devdoot Food Bank donations, corporate ordering, the Sector 49 location and a 2025 founding year. They do not currently identify farms or verify a local-sourcing programme, so the website does not present that as an established fact.
